# Getting Started

This guide gets you from zero to a verified working setup.

## Prerequisites

- Node.js 24+
- pnpm 10+
- GitHub account (for login)

## 1) Install the CLI

```bash
pnpm add -g @tank/cli
# or: npm i -g @tank/cli
```

Verify:

```bash
tank --version
```

## 2) Authenticate

```bash
tank login
```

This opens browser auth and stores your token in `~/.tank/config.json`.

Check identity:

```bash
tank whoami
```

## 3) Install a skill

```bash
tank install @org/skill-name
```

Optional global install:

```bash
tank install @org/skill-name '*' -g
```

## 4) Verify safety and integrity

```bash
tank permissions
tank verify
tank audit @org/skill-name
```

## 5) Create your first skill (publisher path)

```bash
mkdir my-skill && cd my-skill
tank init
```

Then publish safely:

```bash
tank doctor
tank publish --dry-run
tank publish
```

## Success Criteria

You are done when all are true:

- `tank whoami` shows your user
- `tank install` succeeds
- `tank verify` succeeds
- `tank permissions` shows expected access

## Troubleshooting

### Login works but commands fail

Run:

```bash
tank whoami
tank doctor
```

### Install fails on integrity

Re-run install and verify network/proxy stability. Do not bypass verification.

### Permission output looks too broad

Prefer packages with narrow, explicit permission scopes.
