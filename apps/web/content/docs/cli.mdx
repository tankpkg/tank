description: Complete reference for all tank CLI commands

The Tank CLI provides 16 commands for publishing, installing, and managing AI agent skills with security-first design.

## Installation

```bash
npm install -g @tank/cli
# or
pnpm add -g @tank/cli
```

## Global Options

All commands support these options:

| Option | Description |
|--------|-------------|
| `-h, --help` | Display help for the command |
| `-V, --version` | Display the CLI version |

## tank init

Create a new skills.json in the current directory

```bash
tank init
```


## tank login

Authenticate with the Tank registry via browser

```bash
tank login
```


## tank whoami

Show the currently logged-in user

```bash
tank whoami
```


## tank logout

Remove authentication token from config

```bash
tank logout
```


## tank verify

Verify installed skills match the lockfile

```bash
tank verify
```


## tank audit

Display security audit results for installed skills

```bash
tank audit [name]
```

### Arguments

| Name | Description | Required |
|------|-------------|----------|
| `name` | Skill name to audit (omit to audit all) | No |


## tank unlink

Remove skill symlinks from AI agent directories

```bash
tank unlink
```


## tank doctor

Diagnose agent integration health

```bash
tank doctor
```
