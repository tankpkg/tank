# Where to Look

Task-to-location mapping for the Tank monorepo.

| Task                     | Location                                    | Notes                                                          |
| ------------------------ | ------------------------------------------- | -------------------------------------------------------------- |
| Add CLI command          | `packages/cli/src/commands/`                | Export async fn, register in `bin/tank.ts`                     |
| Add MCP tool             | `packages/mcp-server/src/tools/`            | Export `registerXxxTool(server)`, register in `index.ts`       |
| Add API endpoint         | `packages/web/app/api/v1/` or `api/admin/`  | Next.js Route Handler pattern                                  |
| Add UI page              | `packages/web/app/`                         | Route groups: `(auth)`, `(dashboard)`, `(admin)`, `(registry)` |
| Add server action        | `packages/web/app/(dashboard)/*/actions.ts` | Existing: tokens, orgs                                         |
| Modify DB schema         | `packages/web/lib/db/schema.ts`             | Run `drizzle-kit generate` after                               |
| Modify auth schema       | `packages/web/lib/db/auth-schema.ts`        | Auto-generated by better-auth                                  |
| Add shared type/schema   | `packages/shared/src/`                      | Export from `index.ts` barrel                                  |
| Add UI component         | `packages/web/components/ui/`               | `npx shadcn add <component>`                                   |
| Modify permissions model | `packages/shared/src/schemas/permissions.ts`| Zod schema                                                     |
| Auth configuration       | `packages/web/lib/auth.ts`                  | better-auth with GitHub OAuth + OIDC SSO                       |
| Modify security scanner  | `packages/scanner/lib/scan/`                | 6 stages: stage0–stage5                                        |
| Environment variables    | `.env.local` (root)                         | Copy from `.env.example`                                       |
| Performance testing      | `packages/web/scripts/perf-*`               | Seed → analyze → report                                        |
| E2E tests                | `e2e/`                                      | Needs `.env.local` with real credentials                       |
| BDD tests                | `.bdd/steps/`                               | Step definitions, `bun test:bdd`                               |
| Email service            | `packages/web/lib/email/`                   | Rate-limited email via Resend/SMTP                             |
| Storage abstraction      | `packages/web/lib/storage/provider.ts`      | Supabase or S3 backends                                        |
| Admin middleware         | `packages/web/lib/admin-middleware.ts`      | Role-based access control                                      |
| Audit logging            | `packages/web/app/api/admin/audit-logs/`    | Admin action logging                                           |
| GitHub Action            | `action/action.yml`                         | Composite action for CI/CD publishing                          |
| Documentation pages      | `packages/web/content/docs/`                | MDX files (Fumadocs)                                           |
| Data access layer        | `packages/web/lib/data/skills.ts`           | Optimized DB queries                                           |
