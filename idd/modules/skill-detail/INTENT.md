# Skill Detail Page

## What
Full detail view for a published skill — README, versions, file explorer, security report, permissions, install command.

## Why
Primary decision point for installation. Must provide all info to evaluate trust and utility.

## Acceptance Criteria
- [ ] README rendered with syntax highlighting
- [ ] Version selector with changelog
- [ ] File explorer showing package contents
- [ ] Security scan results displayed
- [ ] Permissions listed
- [ ] One-click install command
